# TerraPets Web

Proyecto listo para subir a Vercel.

## Cómo usarlo localmente
1. Instala Node.js
2. Abre terminal en esta carpeta
3. Ejecuta:
   npm install
   npm run dev

## Para subir a Vercel
1. Sube esta carpeta a GitHub o arrástrala en Vercel
2. Vercel detectará Vite automáticamente
3. Luego conectas tu dominio en Settings > Domains
